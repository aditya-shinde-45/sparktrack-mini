# SparkTrack Mini - PBL Management System

**SparkTrack Mini** is a web-based Project-Based Learning (PBL) management system designed for educational institutes. It allows Admins, Mentors, and External users to manage PBL groups, assignments, evaluations, and authentication efficiently.

## Features
- **Role-based authentication**
  - Admin
  - Mentor
  - External user
- **Admin Panel**
  - View and update external users
  - Assign groups
- **Mentor Panel**
  - View assigned groups
  - Manage students
- **External Users**
  - Login with External ID and password
  - View assigned groups
- **JWT-based secure authentication**
- **Search and filter functionality**
- Editable tables for external user management
- Clean UI built with React and Tailwind CSS

---

## Technologies Used
- **Frontend**: React, Tailwind CSS, React Router
- **Backend**: Node.js, Express.js
- **Database**: Supabase (PostgreSQL)
- **Authentication**: JWT
- **Deployment**: Render / Any cloud platform
- **API Requests**: Axios
